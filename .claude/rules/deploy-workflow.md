# デプロイワークフロー

このプロジェクトのgitリポジトリルートは `/Users/ATSUSHITO_RYCE/CLAUDE/posse/`。
`new-app/` が直接gitで管理されているため、コピー不要で直接デプロイ可能。

## デプロイ手順

1. `new-app/` 内のファイルを修正
2. キャッシュバスティング `?v=N` を `?v=N+1` に更新
3. `git add new-app/...` → コミット → `git push origin main`
4. 2-5分待ってブラウザでハードリフレッシュ（Cmd+Shift+R）

## 絶対にやってはいけないこと

- キャッシュバスティングを更新せずにpushすること
- ファイルをコピーしてからコミットすること（旧二重パス構造は廃止済み）
